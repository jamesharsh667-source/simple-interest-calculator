# Simple Interest Calculator

A simple web application that calculates Simple Interest.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Features

- Calculate simple interest instantly
- Easy-to-use interface
- Built using HTML, CSS, and JavaScript

## Input

- Principal Amount
- Rate of Interest
- Time (Years)

## Output

- Simple Interest

## Technologies Used

- HTML
- CSS
- JavaScript
